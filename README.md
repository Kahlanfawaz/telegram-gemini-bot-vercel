# بوت تليجرام يعمل بتقنية Gemini AI

هذا المشروع عبارة عن بوت تليجرام بسيط مكتوب بلغة بايثون، يستخدم مكتبة `python-telegram-bot` للتعامل مع رسائل تليجرام ومكتبة `google-genai` للاتصال بنموذج الذكاء الاصطناعي Gemini.

## المتطلبات الأساسية

1.  **توكن بوت تليجرام (Telegram Bot Token):** يتم الحصول عليه من `@BotFather` على تليجرام.
2.  **مفتاح Gemini API (Gemini API Key):** يتم الحصول عليه من [Google AI Studio](https://ai.google.dev/gemini-api/docs/api-key).
3.  **بيئة بايثون (Python 3.8+).**

## إعداد وتشغيل البوت

اتبع الخطوات التالية لتشغيل البوت على جهازك أو خادمك:

### 1. استنساخ المشروع

```bash
git clone <رابط المستودع> # إذا كان المشروع على مستودع
# أو قم بإنشاء مجلد جديد ووضع الملفات فيه
mkdir telegram_gemini_bot
cd telegram_gemini_bot
```

### 2. إعداد البيئة الافتراضية وتثبيت المكتبات

من الأفضل دائماً استخدام بيئة افتراضية لعزل تبعيات المشروع.

```bash
# إنشاء بيئة افتراضية
python3 -m venv venv

# تفعيل البيئة الافتراضية
source venv/bin/activate

# تثبيت المكتبات المطلوبة
pip install -r requirements.txt
```

**ملاحظة:** ملف `requirements.txt` يحتوي على قائمة المكتبات التي تم استخدامها في هذا المشروع.

### 3. تعيين متغيرات البيئة

يجب تعيين مفتاحي API كمتغيرات بيئة لضمان أمانهما.

```bash
export TELEGRAM_BOT_TOKEN="YOUR_TELEGRAM_BOT_TOKEN"
export GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
```

**تأكد من استبدال `YOUR_TELEGRAM_BOT_TOKEN` و `YOUR_GEMINI_API_KEY` بالقيم الفعلية الخاصة بك.**

### 4. تشغيل البوت

يمكنك تشغيل البوت باستخدام الأمر التالي:

```bash
python3 bot.py
```

لتشغيل البوت في الخلفية بشكل مستمر (على خادم Linux)، يمكنك استخدام `nohup` أو `screen` أو `tmux`، أو استخدام نظام إدارة خدمات مثل `systemd`.

```bash
# مثال على التشغيل في الخلفية باستخدام nohup
nohup python3 bot.py &
```

## هيكل الكود (`bot.py`)

ملف `bot.py` هو الملف الرئيسي الذي يحتوي على منطق البوت:

*   **التهيئة:** يتم قراءة `TELEGRAM_BOT_TOKEN` و `GEMINI_API_KEY` من متغيرات البيئة.
*   **Gemini Client:** يتم تهيئة عميل Gemini باستخدام المفتاح المقدم.
*   **دالة `start`:** تستجيب للأمر `/start` برسالة ترحيب.
*   **دالة `handle_message`:**
    *   تستقبل الرسائل النصية من المستخدم.
    *   ترسل الرسالة إلى نموذج **`gemini-2.5-flash`** للحصول على الرد.
    *   ترسل الرد الذي يولده Gemini مرة أخرى إلى المستخدم.
*   **الدالة `main`:** تقوم بإعداد `Application` من مكتبة `python-telegram-bot` وإضافة معالجات الأوامر والرسائل، ثم تبدأ عملية الاستقصاء (`run_polling`).
